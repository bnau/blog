Ce thème est livré avec l'intégration [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) installée et configurée dans votre fichier `astro.config.mjs`. Si vous préférez ne pas utiliser MDX, vous pouvez désactiver le support en supprimant l'intégration de votre fichier de configuration.

## Pourquoi MDX ?

MDX est une variante spéciale de Markdown qui prend en charge la syntaxe JavaScript et JSX intégrée. Cela permet de [mélanger JavaScript et composants UI dans votre contenu Markdown](https://docs.astro.build/en/guides/markdown-content/#mdx-features) pour des choses comme des graphiques interactifs ou des alertes.

Si vous avez déjà du contenu écrit en MDX, cette intégration devrait faciliter la migration vers Astro.

## Exemple

Voici comment importer et utiliser un composant UI à l'intérieur de MDX.
Lorsque vous ouvrez cette page dans le navigateur, vous devriez voir le bouton cliquable ci-dessous.

import HeaderLink from '../../components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('cliqué !')">
	Composant intégré dans MDX
</HeaderLink>
